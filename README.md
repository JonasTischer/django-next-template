1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your-repo/django-next-template.git
   cd django-next-template
   ```

2. **Start the Docker Containers**:
   ```bash
   docker-compose up
   ```

3. **Access the Application**:
   - Backend: `http://localhost:8000`
